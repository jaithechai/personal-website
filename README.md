# Jai Dilbaghi Portfolio

A portfolio website built with Next.js, TypeScript, Tailwind CSS, and Framer Motion.

## Quick start

```bash
npm install
npm run dev
```

Open http://localhost:3000 in your browser.

## Project structure

```text
src/
├── app/

│   ├── Work.tsx

    └── utils.ts
```
